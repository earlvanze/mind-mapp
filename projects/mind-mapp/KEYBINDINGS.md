# Mind Mapp — Keybindings (v0.1)

## Creation & Editing
- **Enter** → create sibling node
- **Tab** → create child node
- **Shift+Tab** → promote node (move to parent’s level)
- **Delete / Backspace** → delete node
- **F2** or **Double‑click** → edit label

## Navigation
- **Arrow keys** → move focus to nearest node (directional)
- **Cmd/Ctrl+K** → search & jump
- **Esc** → clear selection / close dialogs

## View
- **Space + Drag** → pan
- **Ctrl/Cmd + Wheel** → zoom
- **0** → reset zoom
- **F** → fit to view

## File
- **Cmd/Ctrl+S** → export JSON
- **Cmd/Ctrl+Shift+S** → export PNG
